# هيكلية نظام إدارة المدرسة

## الفكرة الأساسية
يعتمد النظام على فصل "الملف الثابت للطالب" عن "سجل السنة الدراسية". لا يتم ربط درجات الطالب أو صفه بملفه الشخصي مباشرة، بل تُربط بسجل تسجيل (Enrollment) خاص بسنة دراسية محددة.

## الوحدات الرئيسية (Modules)
1. **السنوات الدراسية (Academic Years):** - كل العمليات في النظام مرتبطة بالسنة الدراسية النشطة حالياً.
2. **سجل التسجيل (Enrollments):**
   - هو حلقة الوصل بين الطالب (Student) والسنة الدراسية (Academic Year).
   - يحتوي على: الصف، الشعبة، حالة النجاح/الرسوب.
3. **التقييم (Grades & Exams):**
   - الدرجات والملاحظات تُربط بـ `EnrollmentId` لضمان عدم ضياع درجات السنوات السابقة عند ترقية الطالب.
