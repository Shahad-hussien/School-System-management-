# توثيق قاعدة البيانات (Prisma + Supabase)

## الجداول الأساسية
- **Student**: يخزن البيانات الدائمة (الاسم، التولد، اسم الأم، الإعاقة).
- **AcademicYear**: يخزن السنوات الدراسية (مثل 2025-2026).
- **Enrollment**: الجدول المحوري. يربط `Student` بـ `AcademicYear`.
- **Grade**: تفاصيل الدرجات الشهرية والنهائية، يرتبط بـ `Enrollment`.
- **Document**: لتخزين روابط الصور المرفوعة (هويات، وثائق، أوراق امتحانات).
